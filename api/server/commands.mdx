## Teleport Commands

### /tp

Teleport to coordinates or another player.

<ParamField path="x" type="number | playerId" required>
  X coordinate or player ID to teleport to
</ParamField>

<ParamField path="y" type="number">
  Y coordinate (required with z if using coordinates)
</ParamField>

<ParamField path="z" type="number">
  Z coordinate (required with y if using coordinates)
</ParamField>

**Required Permission:** `group.admin`

```lua
-- Teleport to player ID 1
/tp 1

-- Teleport to coordinates
/tp 100 200 30
```

### /tpm

Teleport to your current waypoint marker.

**Required Permission:** `group.admin`

```lua
/tpm
```

## Server Management

### /togglepvp

Toggle server-wide PVP on or off.

**Required Permission:** `group.admin`

```lua
/togglepvp
```

### /openserver

Open the server to allow players to join.

**Required Permission:** `group.admin`

```lua
/openserver
```

### /closeserver

Close the server and kick non-whitelisted players.

<ParamField path="reason" type="string" required>
  Reason for closing the server
</ParamField>

**Required Permission:** `group.admin`

```lua
/closeserver Server maintenance
```

## Permission Commands

### /addpermission

Add a permission to a player.

<ParamField path="id" type="playerId" required>
  Player server ID
</ParamField>

<ParamField path="permission" type="string" required>
  Permission name to grant
</ParamField>

**Required Permission:** `group.admin`

**Note:** This command is deprecated. Use ACE permissions configuration instead.

```lua
/addpermission 1 admin
```

### /removepermission

Remove a permission from a player.

<ParamField path="id" type="playerId" required>
  Player server ID
</ParamField>

<ParamField path="permission" type="string" required>
  Permission name to revoke
</ParamField>

**Required Permission:** `group.admin`

**Note:** This command is deprecated. Use ACE permissions configuration instead.

```lua
/removepermission 1 admin
```

## Vehicle Commands

### /car

Spawn a vehicle.

<ParamField path="model" type="string" required>
  Vehicle model name
</ParamField>

<ParamField path="keepCurrentVehicle" type="boolean">
  Whether to keep the current vehicle (deletes current if false)
</ParamField>

**Required Permission:** `group.admin`

```lua
-- Spawn vehicle (deletes current)
/car adder

-- Spawn vehicle (keeps current)
/car adder true
```

### /dv

Delete nearby vehicles or the vehicle you're in.

<ParamField path="radius" type="number">
  Radius to delete vehicles in (optional)
</ParamField>

**Required Permission:** `group.admin`

```lua
-- Delete vehicle you're in
/dv

-- Delete all vehicles within 10 meters
/dv 10
```

**Required Permission:** `group.admin`

/setmoney 1 bank 50000

## Communication Commands

### /ooc

Send an out-of-character message to nearby players.

<ParamField path="message" type="string" required>
  Message to send (supports multiple words)
</ParamField>

```lua
/ooc Need to go AFK for a minute
```

**Note:** Only players within 20 meters and admins who have opted in will see the message.

### /me

Display a roleplay action above your character.

<ParamField path="message" type="string" required>
  Action description
</ParamField>

```lua
/me pulls out a flashlight
/me checks the time on their watch
```

## Utility Commands

```lua

```
